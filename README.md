## 텍스트를 입력 받아 알맞는 명언/조언을 해주는 서비스

### **프로젝트 개요**

이 프로젝트는 사용자가 입력한 텍스트를 바탕으로 명언과 조언 2개를 제공합니다. 입력 한 번으로 끝나는 UX가 목표이며 예외 상황(명언 조회 실패, 위기 신호 등)을 정책으로 처리합니다.

### LLM

이 프로젝트에서 LLM은 대화가 아닌 콘텐츠 생성 엔진으로 활용됩니다.

- 번역/의역
    - Qoutes API에서 가져온 영어 명언을 한국어로 가능한 자연스럽게 의역합니다.
- 조언 생성
    - 사용자의 입력을 바탕으로 실행 가능한 조언을 2개 생성합니다.
    - 대화형 AI가 아니기 때문에 단발형 응답을 합니다.
- 포맷 이탈 방지
    - 결과는 항상 명언 2개와 조언 2개를 출력 되게 프롬프트 규칙을 고정하고 후처리 로직을 추가했습니다.
    - 검색으로 확보된 명언을 LLM 프롬프트 컨텍스트에 주입하여 임의가 아닌 텍스트 기반 생성을 유도했습니다.

### RAG

일반적으로 쓰이는 벡터DB 기반 RAG가 아닌 검색 기반 경량 RAG를 사용했습니다.

- 검색 및 조회
    - 사용자 입력을 규칙 기반으로 분석해 주제 키워드를 추출, 이를 영문 쿼리로 매핑한 뒤 Ninjas Quotes API를 검색 호출하여 관련 명언 후보를 확보합니다.
